# Node24 Playwright FFmpeg

Production-ready Docker image with:

- Node.js 24
- Playwright
- Chromium
- FFmpeg
- yt-dlp

## Pull Image

```bash
docker pull tushar9871/node24-playwright-ffmpeg:v1
```

## Use in Dockerfile

```dockerfile
FROM tushar9871/node24-playwright-ffmpeg:v1
```
